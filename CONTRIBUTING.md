# Contributing

Thank you for your interest in improving OwlPay Wallet Pro CLI.

## Reporting Bugs

1. Check [existing issues](../../issues) to avoid duplicates.
2. Use the **Bug Report** template when opening a new issue.
3. Include the output of `owlp --version`, your OS, and Node.js version.
4. Provide clear steps to reproduce the problem.

**Never include private keys, mnemonics, or session tokens in your report.**

## Requesting Features

1. Use the **Feature Request** template.
2. Describe the use case — what are you trying to accomplish?
3. Mention whether the feature is for human use, AI agent use, or both.

## Contributing Skills

The `skills/` directory contains agent skills distributed via `npx skills add`. If you'd like to improve the existing skill or propose a new one:

1. Fork this repository.
2. Make your changes in the `skills/` directory.
3. Open a pull request with a clear description of what changed and why.

## Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.

## Questions?

For general questions, please use [GitHub Discussions](../../discussions) instead of opening an issue.
